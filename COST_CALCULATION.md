# Модуль расчета себестоимости

## 📊 Что добавлено

Новый модуль для детального расчета себестоимости химических наборов с разбивкой по категориям:

- **Реактивы** - 20 химических реактивов
- **Комплектующие** - 15 позиций (пипетки, флаконы, капсулы и т.д.)
- **Печатная продукция** - 7 позиций (коробки, этикетки, методички)
- **Работа** - 3 позиции (розлив, сборка, аренда)

## 🎨 Дизайн

Интерфейс переработан в минималистичном стиле:
- Чистые таблицы без лишних элементов
- Простые кнопки и формы
- Фокус на данных, а не на декоративных элементах
- Монохромная цветовая схема (черный, белый, оттенки серого)

## 🚀 Как запустить

### 1. Импортируйте данные о себестоимости

```bash
cd backend
npm run seed:cost
```

Это создаст все компоненты из вашей Google Sheets таблицы:
- 20 реактивов
- 15 комплектующих
- 7 элементов печатной продукции
- 3 позиции работы
- Набор "Химичка" с итоговой себестоимостью 1124,14₽

### 2. Запустите приложение

```bash
# Если еще не запущено
./start.sh
```

Или вручную:

Терминал 1:
```bash
cd backend
npm run dev
```

Терминал 2:
```bash
cd frontend
npm run dev
```

### 3. Откройте страницу расчета

Перейдите в браузере: http://localhost:5173/cost-calculation

## 📋 Структура данных

### Таблица: components

Все элементы себестоимости хранятся в одной таблице с разными категориями:

```sql
id                  UUID
name                VARCHAR(255)    -- Название
category            ENUM            -- reagent, equipment, print, labor
purchase_cost       DECIMAL         -- Стоимость закупки (для реактивов)
batch_weight        DECIMAL         -- Масса партии в граммах
per_kit_amount      DECIMAL         -- Количество на 1 набор
price_per_gram      DECIMAL         -- Цена за 1 г
unit_price          DECIMAL         -- Цена за единицу
quantity_per_kit    DECIMAL         -- Количество в наборе
price_per_kit       DECIMAL         -- Итоговая цена на 1 набор
is_active           BOOLEAN
created_at          TIMESTAMP
updated_at          TIMESTAMP
```

### Таблица: kits

Информация о наборах с расчетом себестоимости:

```sql
id                  UUID
name                VARCHAR(255)    -- Название набора
sku                 VARCHAR(100)    -- Артикул
batch_size          INT             -- Размер партии
reagents_cost       DECIMAL         -- Стоимость реактивов
equipment_cost      DECIMAL         -- Стоимость комплектующих
print_cost          DECIMAL         -- Стоимость печати
labor_cost          DECIMAL         -- Стоимость работы
total_cost          DECIMAL         -- Итоговая себестоимость
retail_price        DECIMAL         -- Розничная цена
wholesale_price     DECIMAL         -- Оптовая цена
is_active           BOOLEAN
created_at          TIMESTAMP
updated_at          TIMESTAMP
```

## 🔌 API Endpoints

### Компоненты

```
GET    /api/components              # Все компоненты
GET    /api/components/:id          # Один компонент
POST   /api/components              # Создать
POST   /api/components/bulk-import  # Массовый импорт
PUT    /api/components/:id          # Обновить
DELETE /api/components/:id          # Удалить
```

**Фильтры:**
- `?category=reagent` - только реактивы
- `?category=equipment` - только комплектующие
- `?category=print` - только печать
- `?category=labor` - только работа
- `?active=true` - только активные

### Наборы

```
GET    /api/kits                    # Все наборы
GET    /api/kits/:id                # Один набор
GET    /api/kits/:id/calculate      # Рассчитать себестоимость
POST   /api/kits                    # Создать
PUT    /api/kits/:id                # Обновить
DELETE /api/kits/:id                # Удалить
```
