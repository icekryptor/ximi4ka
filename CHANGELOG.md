# Changelog

## [1.1.0] - 2024-02-16

### ✨ Добавлено

- **Модуль расчета себестоимости**
  - Новые сущности: Component, Kit, KitComponent
  - Детальный расчет с разбивкой по категориям
  - Импорт данных из Google Sheets (скрипт seed)
  - API endpoints для управления компонентами и наборами
  - Страница "Себестоимость" с минималистичным интерфейсом

- **Категории компонентов:**
  - Реактивы (20 позиций)
  - Комплектующие (15 позиций)
  - Печатная продукция (7 позиций)
  - Работа (3 позиции)

- Упрощены цвета (монохромная схема)

- Чистые таблицы и формы
- Фокус на данных

### 🔧 Технические изменения

- Добавлены новые entities (Component, Kit, KitComponent)
- Созданы controllers (component.controller, kit.controller)
- Добавлены API routes (/api/components, /api/kits)
- Обновлена конфигурация базы данных
- Создан seed скрипт для импорта данных о себестоимости
- Добавлены API клиенты для фронтенда

### 📚 Документация
