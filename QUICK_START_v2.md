## Запуск (5 минут)

### 1. Установите PostgreSQL (если еще нет)

```bash
brew install postgresql@14
brew services start postgresql@14

# Создайте БД
psql postgres -c "CREATE DATABASE ximfinance;"
psql postgres -c "CREATE USER ximfinance_user WITH PASSWORD 'ximfinance_pass';"
psql postgres -c "GRANT ALL PRIVILEGES ON DATABASE ximfinance TO ximfinance_user;"
```

### 2. Импортируйте данные

```bash
cd backend

# Базовые данные (финансы)
npm run seed

# Данные себестоимости (компоненты)
npm run seed:cost
```

### 3. Запустите приложение

```bash
# Вернитесь в корень проекта
cd ..

# Запустите всё одной командой
./start.sh
```

### 4. Откройте в браузере

```
http://localhost:5173
```

## API Endpoints

### Новые (себестоимость)
- `GET /api/components` - все компоненты
- `GET /api/components?category=reagent` - только реактивы
- `GET /api/kits` - все наборы
- `GET /api/kits/:id/calculate` - расчет себестоимости

### Существующие (финансы)
- `/api/transactions` - транзакции
- `/api/counterparties` - контрагенты
- `/api/categories` - категории
- `/api/reports` - отчеты

## Горячие клавиши

- **Ctrl+R** - обновить данные
- **Ctrl+K** - поиск

## 🆘 Проблемы?

### Backend не запускается
```bash
# Проверьте PostgreSQL
brew services list

# Перезапустите если нужно
brew services restart postgresql@14
```

### Нет данных на странице себестоимости
```bash
cd backend
npm run seed:cost
```

### Порт занят
Измените порты в:
- `backend/.env` → PORT=3002
- `frontend/vite.config.ts` → port: 5174
